## Requirements :
```
1. Node JS
2. MongoDB
3. Redis Server
```
## Usage

1. Install dependencies.

```npm install```

2. setup env for prod or dev:

```cp exaple.env .env```

3. start the server:

```npm run dev``` or ```npm run start```
